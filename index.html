Inspired by

The story behind this project, it is based on experience, mostly when you have to consult a

doctor and he/she prescribe the medicine some time we get a hard time to get exactly prescribeda

medicine. So to support i had this idea, what if, we have some way to search for a medicine via

a web application, in return we get the list of pharmacy that has that particular medicine,

instead of walking all corner of the city in all pharmacy, now we have few name to check with,

before we go there we can call and ask if they have what is needed.
